console.fu

hAlp!

a quick hack to intercept the console.X functions and stop them from spamming your console.

Go to the page's console that your debugging, and add keywords to its blacklist, then intercept the console.X functions of your choice ('error', 'log', 'warn' ) and that's it. if you want a nice UI here, implement it

Uses window.localStorage so lists are persistent but unique per each hostname

  • cfu.add_to_blacklist("keyword") to add string keyword to the blacklist
  • cfu.intercept_fn('error') Intercept the console.X function by passing a string of it's name ['error', 'log', 'warn']
  • cfu.restore_fn('error') Restores the original console.error function by passing a string of it's name ['error', 'log', 'warn']
  • cfu.remove_from_blacklist("keyword") remove from blacklist
  • cfu.get_blacklist() return an array of all the blacklisted keywords
  • cfu.clear_blacklist() to clear the blacklist
